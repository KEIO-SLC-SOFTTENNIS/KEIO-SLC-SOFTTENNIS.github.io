# 慶應SLCソフトテニス
慶應義塾大学唯一の公認ソフトテニスサークル「SLCソフトテニス部」の公式サイトです。

## フォルダ構成

- `index.html` … トップページ
- `pages/` … サブページ（活動報告・年間スケジュール・メンバー紹介）
- `assets/css/main.css` … 共通スタイルシート
- `assets/js/main.js` … 共通スクリプト（サイドバー開閉など）
- `assets/images/` … 画像一式（hero / icons / schedule / members / activities）

## 更新の流れ

1. 新しい画像は `assets/images` 以下に保存し、英数字のみのファイル名を付けてください。
2. サブページを追加するときは `pages/` に HTML を作成し、必要に応じて各ページのサイドバーリンクを更新します。
3. 活動報告の年度が増えたら `pages/activities/` にファイルを追加し、一覧にもリンクを追記してください。

GitHub Pages ではこのままルートに配置すれば公開できます。
